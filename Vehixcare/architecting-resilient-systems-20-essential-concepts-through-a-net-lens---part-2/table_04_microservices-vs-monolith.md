# table_04_microservices-vs-monolith


| Aspect | Monolith | Microservices |
|--------|----------|---------------|
| Deployment | Single unit | Independent |
| Scalability | Whole application | Per service |
| Development | Single codebase | Multiple repositories |
| Data | Single database | Database per service |
| Communication | In-memory calls | Network calls |
| Complexity | Lower initial | Higher initial |
