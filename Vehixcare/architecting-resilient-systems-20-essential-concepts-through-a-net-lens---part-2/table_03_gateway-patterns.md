# table_03_gateway-patterns


| Pattern | Description | Use Case |
|---------|-------------|----------|
| **Edge Gateway** | Single entry point for external clients | Mobile/Web applications |
| **Internal Gateway** | Service-to-service communication | Microservice mesh |
| **BFF (Backend for Frontend)** | Client-specific gateways | Different client types |
| **API Composition** | Aggregate multiple services | Complex UI pages |
