# Gemini Image Prompt

## Microsoft Style - Light Color Theme

---

### Prompt:

```
Create a modern, professional hero image in Microsoft Fluent Design style with a clean, light color palette (white background, soft gradients of blue #EFF6FF and gray #F9FAFB). The image should showcase .NET 10 LINQ technologies as interconnected visual elements.

Central visual: A large, elegant 3D cube or isometric diagram with "LINQ" prominently displayed on the front face. The cube should be composed of floating, interconnected nodes representing key LINQ method categories: Filtering, Projection, Aggregation, Joining, Set Operations, Quantifiers, Generation, and Ordering. Each category appears as a glowing blue tile with method icons.

Surrounding elements: Abstract representations of performance features - SIMD vectorization sparkles (blue light bursts), NativeAOT compilation symbol (a green checkmark shield), EF Core database cylinder with SQL translation arrows, and async stream flow lines (wavy cyan lines). All elements have subtle drop shadows and soft gradients.

Background: Soft white with circular gradient overlays in Azure blue (#0078D4) at 5% opacity, creating depth without distraction.

Typography: 
- Title: "LINQ Reimagined: A Deep Dive into .NET 10's Query Evolution" in bold, clean sans-serif font (Segoe UI or Inter), dark navy blue (#1E3A8A), large and centered.
- Subtitle: "Performance Revolution | SIMD | NativeAOT | Async | EF Core" in smaller text below title, medium weight, in gray (#4B5563).

Layout: Title and subtitle positioned at top center. Visual elements arranged in an arc below them, leading the eye toward the bottom right corner.

Bottom right corner: "Vineet Sharma" in elegant, professional signature font, dark blue (#1E40AF), with a subtle line separator above.

Overall style: Microsoft Fluent Design, clean, minimalist, technical yet approachable, light academia meets developer tools aesthetic. High resolution, 16:9 aspect ratio, suitable for blog hero or social media banner. Low resolution
```
