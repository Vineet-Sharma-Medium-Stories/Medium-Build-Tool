# table_02_ide-specific-features


| IDE | Key Integrations |
|-----|------------------|
| **VS Code** | Native integration, Copilot Chat sidebar, inline chat, Workspace Commands |
| **JetBrains** | Full IDE support, chat window, code inspection integration |
| **Visual Studio** | Enterprise-focused, integrated with .NET workflows |
| **Neovim** | Terminal-based, lightweight plugin |
| **Eclipse** | Java-focused, enterprise support |
| **Xcode** | Beta support for iOS/macOS development |
