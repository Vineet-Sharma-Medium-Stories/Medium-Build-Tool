# Gemini Image Generation Prompts

## The Complete Multi-Language Containerization Library: .NET, Python, and Node.js on Azure and AWS

---

### Prompt 1: Main Banner - Complete Multi-Language Containerization Library

```markdown
Create a wide horizontal banner image (16:9 aspect ratio) for "The Complete Multi-Language Containerization Library" with a light, open-source style background. The design should feature:

**Background:**
- Light gradient from off-white (#f8f9fa) to light gray (#e9ecef)
- Subtle grid pattern suggesting infrastructure/architecture
- Soft shadow effects for depth

**Central Elements:**
- Three distinct columns representing .NET, Python, and Node.js
- Each column topped with official logos:
  - Left: .NET logo (purple/magenta) with "10" badge
  - Center: Python logo (blue/yellow) with "FastAPI" text
  - Right: Node.js logo (green) with "Express.js" text

**Cloud Platform Connectors:**
- Azure logo (blue) on left side with curved connector lines flowing to all three language columns
- AWS logo (orange) on right side with curved connector lines flowing to all three language columns
- Connector lines should be dotted or dashed with subtle animation suggestion

**Bottom Section:**
- 10 icons in a horizontal row representing the 10 deployment approaches:
  1. SDK Native (cube icon)
  2. Dockerfile (container icon)
  3. Podman (leaf icon)
  4. Serverless (cloud with arrow)
  5. IDE (monitor with code)
  6. Cloud CLI (terminal icon)
  7. Security (shield icon)
  8. Hybrid (merge icon)
  9. Modern Tools (rocket icon)
  10. Kubernetes (helm icon)

**Text Elements:**
- Main title: "The Complete Multi-Language Containerization Library" in bold, modern sans-serif font
- Subtitle: ".NET 10 • Python FastAPI • Node.js Express" below main title
- "Azure" and "AWS" labels near respective cloud logos
- "60 Deployment Stories" as a badge in the corner

**Attribution:**
- **"Vineet Sharma" in bold font at the bottom right corner, placed in a clean area not overlapping with any icons, text, or logos. Use a subtle, elegant font with slight opacity or a minimal badge style to ensure visibility without distracting from the main design.**

**Style:**
- Clean, modern open-source aesthetic
- Light background with subtle gradients
- 3D elements with soft shadows
- Flat design with slight depth
- High resolution (1920x1080 or larger)
```
