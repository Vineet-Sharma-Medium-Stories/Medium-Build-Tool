# ### Cost-Saving Strategies for GCP

| Strategy | Savings | Implementation |
|----------|---------|----------------|
| **Scale to zero** | 80-95% | Cloud Run min-instances=0 |
| **Committed use discounts** | 20-40% | 1-year commitment for GKE |
| **Preemptible VMs** | 60-80% | Use for batch processing |
| **Coldline storage** | 50-70% | Archive old logs to Coldline |
| **Regional vs Multi-regional** | 30-50% | Use regional storage |
