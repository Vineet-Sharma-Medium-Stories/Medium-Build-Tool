# ### Project Ideas

| Express Line | Foundation Project |
|--------------|-------------------|
| LLMs & Agents | Build a text classifier from scratch using a small transformer. Train it. Evaluate it. Document the data cleaning steps. |
| Computer Vision | Build an image classifier on CIFAR-10 or Fashion-MNIST. Show data augmentation, training curves, and failure cases. |
| Production ML | Build a training pipeline that loads data, trains a model, logs metrics, and saves artifacts. Use MLflow or similar. |
